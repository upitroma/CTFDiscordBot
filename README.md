# CTFDiscordBot
a bot to help manage the ctf

## how to setup
1. install node.js
2. add a file named `token.json` to the root
3. add your token and logins to token.json like this 
```json
{
    "TOKEN":"your_secret_token_here"
}
```
4. configure flags and responses in `strings.json`
5. run `node index.js`
